# Appell — знаем в лицо

Одностраничный сервис в тёмной теме с эффектом «жидкого стекла». Чат с нейросетью: загрузка скриншотов и видео, анализ и вывод списка никнеймов игроков.

## Запуск

1. Установи зависимости:
   ```bash
   npm install
   ```

2. Задай ключ OpenAI (для анализа изображений):
   ```bash
   export OPENAI_API_KEY=sk-...
   ```

3. Запусти сервер:
   ```bash
   npm start
   ```

4. Открой в браузере: [http://localhost:3000](http://localhost:3000)

## Как пользоваться

- Ввести текст и/или прикрепить изображения/видео.
- Для видео с фронта отправляется один кадр; нейросеть ищет ники на кадре.
- Ответ показывается в чате; список никнеймов выводится отдельным блоком под сообщением.
